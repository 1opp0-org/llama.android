# Product Definition: Llama Android

## Goal
Provide a high-quality, local-only LLM inference library for Android using `llama.cpp`.

## Core Values
- **Local-only:** No internet required for inference, so it's privacy first.
 - internet allowed for downloading models
- **Scope** android phones
  - initial focus on CPU inference
  - Vulcan is supported by llama.cpp, but need work for android support
- **Quality:** High engineering standards and robust API.
